# Instalação no terminal Linux ?

Abra o terminal é excute os seguintes comamndos: 

* `git clone https://github.com/Pemakerin/YouTube.git`

* `cd YouTube`

* `chmod +x install.sh`

* `bash install.sh`

* `python3 frame_v3.py`
